# Travel
旅遊預訂系統 (Travel Booking System)


A. 系統登入與註冊

輸入帳號密碼後，可透過點選 **「顧客」** 或 **「員工」** 單選鈕來決定登入身份。

● 顧客身份：進入前台預訂系統。

● 員工身份：進入後台管理系統。

若無顧客帳號，可點擊 **「尚未加入會員？點此註冊」** 按鈕，建立一組新的一般會員帳號。

---

B. 前台：顧客行程預訂

登入顧客帳號後，將進入旅遊行程預訂介面，流程如下：

1.  篩選與選擇行程：

    ● 選擇縣市：從下拉選單中選擇想去的縣市。
    
    ● 選擇行程：系統會根據所選縣市，自動載入該地區所有可預訂的旅遊產品至行程下拉選單。
    
    ● 選擇日期與人數：使用日曆元件選擇出遊日期，並手動填寫旅遊人數。

3.  訂單建立：

    ●   按下 「確認訂單」 按鈕，系統會進行欄位驗證。
    
    ●   驗證通過後，將訂單資料寫入資料庫，並跳轉至訂單成功頁面。

5.  訂單成功畫面：
6.  
    ●   清楚展示本次訂單的所有詳細資訊，包含：訂單編號、顧客姓名、產品名稱、旅遊日期、人數及總金額。

7.  返回首頁：
8.  
    ●   點擊 「返回登入首頁」 按鈕，可登出目前帳號並回到登入畫面。

---

C. 後台：員工管理系統

登入員工帳號後，可進入後台主選單，選擇進行「產品管理」或「訂單管理」。

1. 旅遊產品管理 (CRUD)

●   瀏覽 (Read)：介面會自動載入並列表顯示所有旅遊產品的 **ID、名稱、目的地、價格與描述**。

●   新增 (Create)：在上方的輸入欄位填寫新產品的完整資訊後，點擊 **「新增產品」** 按鈕即可上架。

●   修改 (Update)：

    1.  在 「操作目標 ID」 欄位輸入要修改的產品 ID。
    
    2.  在上方的輸入欄位填入新的產品資訊。
    
    3.  點擊 「確認修改」 按鈕完成更新。

●   刪除 (Delete)：在 「操作目標 ID」 欄位輸入要刪除的產品 ID，點擊 「確認刪除」 按鈕即可將產品下架。


2. 顧客訂單管理

●   訂單查詢 (Read)：介面會自動載入所有歷史訂單紀錄，並以格式化的表格顯示 訂單 ID、顧客姓名、預訂產品、日期、人數與總價。

●   刪除訂單 (Delete)：在下方的 「輸入要刪除的訂單 ID」 欄位輸入訂單 ID，點擊 「確認刪除」 即可移除該筆訂單紀錄。
